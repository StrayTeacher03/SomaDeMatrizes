# Soma das Matrizes

## Descrição:
Programa que lê 2 Matrizes bidimensionais e as soma e guarda em uma matriz resultado, tem operações como:

Ler a Matriz

Fazer a soma

Exibir as operações



## Como executar:
1- Compile o programa

2- Execute a classe principal na pasta app

## Exemplos de uso:
Entrada:
3, 4, 4, 4

5, 6, 6, 8

Saída:

Matriz 1: 3, 4, 4, 4

Matriz 2: 5, 6, 6, 8

Resultado: 8, 10, 10, 12
